# Hurrican Harvey Refugee Helper

## Overview

A simple React app:

* Radium (CSS)
* Redux (store)
* Webpack (compilation)

## Versions
* **React**: `15.6.1`
* **Node**: `5.10.1`

## Getting Started

```
npm install -g webpack webpack-dev-server
nvm use
npm run start
```

Visit `localhost:8080`

## Build

```
npm run build
```
